# Работа — агрегация вакансий: roadmap и планы по странам

> Артефакт модуля «Работа» (job aggregator).
> Дата: 2026-06-29.
> Канон: `docs/work-aggregator-roadmap.md` в репозитории `avalone.online`.

---

## 1. Что это

Job-агрегатор внутри Avalone собирает открытые вакансии из внешних job-board'ов, нормализует их в единую схему, переводит контент и показывает пользователям через единый интерфейс `/work`.

Продуктовый фокус:
- **Низкий порог входа** — подработки, ежедневная оплата, арбайт, gig-работа.
- **Стабильность** — официальное трудоустройство, полная занятость.
- **Иностранцы + носители языка** — не дискриминируем по паспорту; фильтры помогают каждому найти подходящее.
- **Выбор страны и региона** — пользователь может жить в одной стране, а искать работу в другой.

## 2. Текущая архитектура и что уже заложено

### 2.1. Модель данных

Таблица `work_job_posts` (единая БД Avalone, `~/.avalone/avalone.db`):
- `external_guid` — уникальный идентификатор источника + внешний ID.
- `source_site`, `source_url` — откуда пришла вакансия.
- `country` — ISO-код страны (например, `KR`).
- `title`, `description_html`, `description_text` — оригинальный контент.
- `title_translated`, `description_translated` — перевод.
- `employer`, `location`, `job_type`, `salary`, `pay_type`.
- `contact_phone`, `contact_email`, `visa_type` — извлекаются из текста.
- `content_hash` — SHA-256 от ключевых полей; защищает от дублирования одинаковых публикаций.
- `posted_at`, `parsed_at`.
- `raw_json` — сырые данные источника для observability.

### 2.2. Парсеры

Реестр парсеров — модульный. Каждый парсер наследуется от `BaseJobParser`, задаёт `country` и реализует `parse(html_text)`.

Текущие источники для Южной Кореи (`KR`):
| Источник | Тип | Примечание |
|----------|-----|------------|
| `KoreabridgeRSSParser` | RSS | Англоязычные вакансии для иностранцев. |
| `AlbamonParser` | Next.js `__NEXT_DATA__` | Подработки и арбайт, ~55% рынка альбы. |
| `SaraminParser` | SSR BeautifulSoup | Полноценные вакансии. |
| `JobKoreaParser` | RSC `self.__next_f.push` | Полноценные вакансии. |
| `OneOneFourParser` | JSON API (`/api/posts`) | Вакансии для иностранцев и производства (114114구인구직). |

### 2.3. Дедупликация и обновления

`JobPostRepository.save()` возвращает статус:
- `inserted` — новая вакансия.
- `updated` — контент изменился; существующие `posted_at` и переводы сохраняются.
- `unchanged` — контент идентичен; обновляется только `parsed_at` и `raw_json`.

Это позволяет запускать парсинг часто, не переписывая БД одинаковыми записями.

### 2.4. UI / фильтры

`/work` поддерживает:
- поиск по тексту (`q`);
- локация (`location`);
- источник (`source`);
- тип визы (`visa`);
- тип работы (`job_type`);
- страна (`country`);
- период (`days`);
- пагинация (20 на страницу).

### 2.5. Перевод

- `scripts/translate_jobs.py` — фоновый переводчик через локальный Kimi CLI.
- Работает пакетами (`--batch`), цепочкой: перевёл один пакет → сразу следующий.
- Ограничения: `--limit`, `--timeout-seconds`, `--max-age-days`, `--max-failures`.
- Режим `--loop` — для демона, который засыпает между проверками.

## 3. Принцип расширения на новые страны

Добавление страны — это добавление одного или нескольких парсеров:

1. Создать класс `<Source>Parser(BaseJobParser)` в `src/avalone_landing/core/jobs/parser.py`.
2. Указать `country = "<ISO>"` (например, `"VN"`, `"US"`).
3. Реализовать `parse(html_text)` → `list[JobPost]`.
4. Добавить парсер в `MultiSourceParser`.
5. Обновить `country` filter в UI — он уже работает по значениям из БД.
6. При необходимости добавить визовые паттерны и location keywords в `JobPostService`.

Не нужно менять схему БД, роутер или фильтры.

## 4. Планы по странам

### 4.1. KR — Южная Корея (текущий фокус)

**Цель:** покрыть основные источники для иностранцев и носителей языка.

**Источники в работе / запланированные:**
- Koreabridge RSS ✓
- Albamon ✓
- Saramin ✓
- JobKorea ✓
- 114114구인구직 (`114114.co.kr`) ✓

**Что нужно доделать по Корее:**
- Добавить второй популярный источник (см. выше).
- Улучшить извлечение контактов и виз из корейских описаний.
- Добавить категорию «ежедневная оплата / еженедельная оплата».

### 4.2. VN — Вьетнам

**Почему:** большая экспат- и мигрантская аудитория, корейские заводы во Вьетнаме, русскоязычные специалисты в IT и производстве.

**Потенциальные источники:**
- VietnamWorks
- CareerBuilder Vietnam
- JobStreet Vietnam
- TopCV.vn
- Indeed Vietnam
- Вьетнамские Telegram/FB-группы (возможно, позже).

**Фокус:** производство, IT, гостиничный бизнес, удалёнка.

### 4.3. US — США

**Почему:** gig economy, сезонная работа, программы J-1 / H-2B, большой рынок для мигрантов.

**Потенциальные источники:**
- Indeed US
- Craigslist (разделы городов)
- Snagajob
- CoolWorks (сезонная работа)
- SeasonalJobs.dol.gov (официальный реестр)
- ZipRecruiter

**Фокус:** сезонная работа, рестораны, складская логистика, retail.

### 4.4. KZ / RU / CIS

**Почему:** русскоязычные мигранты, удалённая работа, релокация.

**Потенциальные источники:**
- HeadHunter (hh.ru / hh.kz)
- Habr Career
- Geekjob
- Weblancer
- FL.ru
- Kolesa.kz (раздел работы)

**Фокус:** удалёнка, IT, рабочие специальности.

### 4.5. JP — Япония

**Почему:** растущий спрос на иностранных работников через Specific Skilled Worker visa.

**Потенциальные источники:**
- GaijinPot Jobs
- Daijob
- JobsInJapan
- Yolo Japan
- HelloWork (официальный, позже)

**Фокус:** фабрики, рестораны, уход, отели.

### 4.6. TH — Таиланд

**Почему:** туризм, удалёнка, растущая экспатская база.

**Потенциальные источники:**
- JobsDB Thailand
- ThaiCV
- FarangMart
- Bangkok Post Jobs

**Фокус:** туризм, гостиницы, IT, удалёнка.

### 4.7. PH — Филиппины

**Почему:** BPO, customer support, удалённая работа.

**Потенциальные источники:**
- JobStreet Philippines
- Indeed PH
- OnlineJobs.ph
- Kalibrr

**Фокус:** BPO, виртуальные ассистенты, IT.

### 4.8. ID — Индонезия

**Почему:** производства, корейские и японские заводы, большой рынок труда.

**Потенциальные источники:**
- JobStreet Indonesia
- Kalibrr
- Indeed Indonesia
- Glints

**Фокус:** производство, логистика, retail.

## 5. Приоритеты

1. **KR** — довести до стабильного ежедневного наполнения (5+ источников, перевод, фильтры).
2. **VN** — следующий рынок: близок географически, много корейских компаний, интересен русскоязычным.
3. **US** — сезонная и gig-работа, высокий интерес мигрантов.
4. **JP, KZ/RU/CIS** — параллельно по мере появления контента.
5. **TH, PH, ID** — на очереди после отработки механики.

## 7. Ограничения и политика

- Используем только открытые, публично доступные данные.
- Права на оригинальные тексты принадлежат авторам/площадкам; Avalone выступает агрегатором и переводчиком.
- Соблюдаем robots.txt и rate limits источников.
- Не храним персональные данные соискателей без explicit consent.
